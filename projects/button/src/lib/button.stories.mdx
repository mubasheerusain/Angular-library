# Button

Custom `Button` Component For CXOne Shared UI. 

## Primary 

`Primary` is the Default variant for this button component. And also their are other available variant like `Basic`, `Text` and `Warn`. The Button also comes with a variable sizing feature like `Large`, `Medium` and `Small`.

<Canvas>
    <Story
    name="Primary"
    args={{
        variant: 'primary',
        disabled: false
    }}>
    {Template.bind({})}
    </Story>
</Canvas>

## Basic 

`Basic` is another variant available in the button. It comes with a light background with oulined feel.

<Canvas>
<Story
   name="Basic"
   args={{
     variant: 'basic',
    disabled: false
   }}>
   {Template.bind({})}
</Story>
</Canvas>

## Text 

`Text` variant comes with a transparent background without border and light shade on hover.
